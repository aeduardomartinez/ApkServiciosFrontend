# ApkServiciosFrontend
# StreamManager App 📱  Aplicación móvil desarrollada con **Ionic + Angular** para la gestión y  comercialización de cuentas de servicios de streaming compartidos  (Netflix, Spotify, Disney+, entre otros).
# StreamManager App 📱


## ¿Qué hace?

Permite a revendedores de streaming administrar sus cuentas y controlar 
qué cliente tiene asignado cada perfil, por cuánto tiempo y en qué estado se encuentra.

## Módulos

- **Servicios** → Configuración de plataformas con precios y límite de perfiles
- **Cuentas** → Registro de cuentas maestras con generación automática de perfiles
- **Clientes** → CRUD de clientes para asignar a los perfiles
- **Perfiles** → Asignación, edición y liberación de cupos por cuenta

## Stack

- Ionic 7
- Angular 17 (Standalone Components)
- Arquitectura Hexagonal (Use Cases + Repository Pattern)
- Programación Reactiva (RxJS)
